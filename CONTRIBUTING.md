# Contributing to LinkedIn Fixer

Thanks for your interest in contributing! Here's how to get started.

## Getting Started

1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/linkedin-fixer.git`
3. Create a branch: `git checkout -b my-feature`
4. Make your changes
5. Push and open a PR against `main`

## Development Setup

```bash
pip install flask flask-cors requests beautifulsoup4 pymupdf
ollama pull llama3.2
python app.py
```

## Guidelines

- Keep PRs focused — one feature or fix per PR
- Test your changes locally before submitting
- Update README if you add new features
- Be respectful in reviews and discussions

## What we'd love help with

- Support for more file formats (docx, txt)
- Better JSON parsing for edge cases
- Dark mode
- Multiple language support
- Deploy to web (Vercel, Railway)
- Chrome extension version

## Reporting Bugs

Open an issue with:
- What you expected to happen
- What actually happened
- Your OS and Python version
